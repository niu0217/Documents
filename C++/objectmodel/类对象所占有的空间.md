# 类对象所占有的空间

## 1. 空类

```c++
#include<iostream>
using namespace std;

class A {
public:
};

int main()
{
    A a;
    cout<<sizeof(a)<<endl;
}

//输出1，表示空类对象至少占一个字节的空间
```

## 2. 类只有成员函数

```c++
#include<iostream>
using namespace std;

class A {
public:
    void func1() {

    }
    void func2() {

    }
    void func3() {

    }
};

int main()
{
    A a;
    cout<<sizeof(a)<<endl;
}

//输出1
//结论：类的成员函数不占用类对象的内存空间
```

## 3. 类只有成员函数和成员变量

```c++
#include<iostream>
using namespace std;

class A {
public:
    void func1() {

    }
    void func2() {

    }
    void func3() {

    }
    int value;
};

int main()
{
    A a;
    cout<<sizeof(a)<<endl; //4，说明类的成员变量是占类对象的空间的

    a.value = 12;
    return 0;
}
```
