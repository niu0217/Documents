# 简单使用

## 检查内存泄漏

代码：

```c
/* ************************************************************************
> File Name:     main1.c
> Author:        niu0217
> Created Time:  Fri 01 Mar 2024 05:15:05 PM CST
> Description:
 ************************************************************************/

#include<stdio.h>
#include<stdlib.h>

int main()
{
    const int size = 100;
    int n;
    int sum = 0;
    int* A = (int*)malloc(sizeof(int) * size);
    for(n = size; n > 0; n--) {
        A[n] = n; //A[100]无效
    }
    for(n = 0; n < size; n++) {
        sum += A[n]; //A[0]没有初始化
    }
    printf("sum = %d\n", sum);
    return 0;
}
```

终端输入命令：

```bash
gcc -g -o main1 main1.c

valgrind --tool=memcheck --leak-check=yes ./main1
```
